这是利用VUE封装的天地图的组件

https://cli.vuejs.org/guide/
##  2,VUE官网
https://cn.vuejs.org/

+ api文档地址――https://dafrok.github.io/vue-baidu-map/#/zh/index

# 运行步骤
1. 下载必要的依赖
npm install
2. 启动项目
npm run serve
